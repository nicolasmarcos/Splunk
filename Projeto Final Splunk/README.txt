EN - This directory contains the files those refers to the final project, used as examination of the Infrastructure's discipline in the Specialization of MIT in Big Data at Instituto Infnet. There are the technical report and the presentation of the study done in the Splunk tool, just as video tutorials of the tool. All the study is in Portuguese. This project was done by: Allan Andrade, Fábio Domingues and Nicolas Marcos. All images were collected through public web sources. None are self-authored.
